# Electro Shop Clone

Clone d’un site e-commerce réalisé dans un objectif pédagogique.

## Description
Ce projet est un clone d’un site e-commerce développé pour renforcer mes compétences
en développement web (front-end et notions full-stack).

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## Fonctionnalités
- Interface e-commerce responsive
- Pages produits
- Navigation utilisateur
- Structure claire du code

## Objectif du projet
- Pratique du développement web
- Maîtrise de Git et GitHub
- Préparation aux projets full-stack (MERN)
